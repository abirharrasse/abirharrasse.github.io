# Blog Posts

Use this folder for standalone blog posts.

To add a new post:

1. Copy `_template.html` to a new descriptive filename, for example `shared-multilingual-representations.html`.
2. Update the `<title>`, `<h1>`, date, and article text in the new file.
3. In `index.html`, find the `<!-- BLOG -->` section and replace the matching placeholder `href="#"` with `href="blogs/your-file-name.html"`.
4. Change the blog date from `coming soon` to the publication date when ready.

Keep filenames lowercase with hyphens so the links stay clean on GitHub Pages.
